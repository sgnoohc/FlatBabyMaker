# Flat Baby Maker


This loops over CMS3 (or CMS4) and create a truly flat baby.

CMS3 looper template.
It uses Ditto.

    # setup ROOT
    mkdir cms3looper
    cd cms3looper
    git clone git@github.com:sgnoohc/FlatBabyMaker.git
